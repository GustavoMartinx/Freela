# Para rodar o backend:

- `npm install`

- Criar um arquivo ".env" neste diretório com as especificações da conexão com o banco de dados:

    - DB_DATABASE = 'Freela'
    - DB_USER = 'root'
    - DB_PASSWORD = 'root'
    - DB_HOST = 'localhost'
    - DB_PORT = 3306
    
- `nodemon server.js`

# Para criação e teste do banco de dados:

- Utilizar os arquivos no diretório "queries"
